# Fundamental Booster Project

The Fundamental Booster Project is a beginner-friendly Python application that collects user information through an interactive console interface. The program asks for details such as name, age, height, and favorite number, then calculates the user's approximate birth year and displays information about the entered data, including data types and memory addresses.

This project was built to practice core Python concepts such as variables, user input, data types, type conversion, f-strings, the `type()` function, and the `datetime` module.

## Technologies Used

* Python
* datetime module

## How to Run

```
python untitled7.py
```

## Learning Outcomes

Through this project, I learned how to collect user input, work with strings, integers, and floats, perform basic calculations, use Python modules, and format output using f-strings.

## Future Improvements

Future versions of this project may include a BMI calculator, input validation, data storage using files, and a graphical user interface (GUI).
