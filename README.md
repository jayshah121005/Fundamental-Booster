# Fundamental Booster Project

## Overview

The Fundamental Booster Project is a simple Python program that collects user information through an interactive interface.

This project was created to practice Python fundamentals such as:

1. Variables
2. User Input
3. Data Types
4. Type function
5. f-Strings

## Features

1. Collects user's name
2. Collects user's age
3. Collects user's height
4. Collects user's favorite number
5. Calculates approximate birth year
6. Displays data types of variables using type()
7. Displays memory addresses using `id()`

## Technologies Used

1. Python Language
2. datetime module

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Navigate to the project folder:
   cd fundamental-booster-project

3. Run the program:
   python untitled7.py
